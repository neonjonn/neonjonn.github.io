The micro:bit

The micro:bit (also known as the BBC micro:bit) was a campaign launched in 2015 to help school children in the UK learn about coding

There are two versions of the micro:bit, the micro:bit V1 and the micro:bit V2

The main differences between the V1 and the V2 is that the V2 has a faster processor and more RAM. The V2 also a microphone and a speaker

Irish Sign Language (ISL)

The Irish sign language is called ISL.

Only 40,000 people in Ireland know ISL

How it Works

We came up with the idea of a device that could translate ISL into English.

Our code is implemented in 99% home-grown cage-free typescript, with a small dash of C++.

  1. For our prototype we have attached a micro:bit to the thumb, it should be able to differentiate between the signs B and L.
  2. Using machine learning on the edge, we detect different positions via a dataset.
  3. We make use of ~100 lines of typescript to make a lexer and parser for custom configuration file format inspired by YAML
  4. We have an automated configuration provisioning tool, configtool.
  5. Our team added a custom C++ binding to the C++ storage API.
  6. In the future we hope to improve on this device and make it translate different sign languages, not just ISL.
